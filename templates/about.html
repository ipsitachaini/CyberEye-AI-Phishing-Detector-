About Phishing Detector

Multi-Modal Phishing Detection System

Advanced cybersecurity protection using artificial intelligence and machine learning to detect phishing attempts across multiple media types.

Features
  • Text Analysis: Detects phishing in emails, messages, and documents using NLP models
  • Image Analysis: Extracts and analyzes text from images using OCR technology
  • Audio Analysis: Transcribes and analyzes audio files for phishing content
  • Video Analysis: Processes video files to detect phishing attempts
  • Real-time Protection: Instant analysis with percentage-based safety scores
Technology Stack
  • Backend: Python Flask
  • AI/ML: Transformers, PyTorch, Scikit-learn
  • Database: SQLAlchemy (SQLite)
  • OCR: Tesseract
  • Audio: Whisper OpenAI
  • Frontend: Bootstrap 5
How It Works
  1. Input Processing: Upload text, images, audio, or video files
  2. Content Extraction: Extract analyzable content using appropriate tools
  3. AI Analysis: Process content through machine learning models
  4. Risk Assessment: Generate safety percentage and classification
  5. Result Delivery: Present clear, actionable results to users
